#1: Green Tread's online store

Sep 2015- Jan 2016

• Project name: Tyre Buyers' Paradise
• Challenge: How can an Estonia-based tyre company attract sales in Finland?
• Role: Examine the web store and find out ways it can be improved from UX point of view.
• Team: 1 UX designer, 2 marketers, 2 programmers.
• Result: First level license from the company.

THE CHALLENGE

Lehvid.com is an Estonia-based tyre web store that belongs to Green Tread Oy in Finland.
The purpose of the project was to improve the current web store in order to attract customers and increase sales via the web store.
My job as a UX designer in the team was to examine, evaluate, and design the current web store accordingly.
